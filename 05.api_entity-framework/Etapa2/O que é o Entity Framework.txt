O Entity Framework é uma ferramenta de mapeamento de relação entre objetos 
(Object Relational Mapping, ou ORM) criado pela Microsoft para facilitar a 
integração entre o back-end e o banco de dados e permite a criação de consultas
SQL na linguagem de programação C#. 

Basicamente, o EF permite que uma tabela em um banco de dados relacional seja 
traduzida para um objeto em C#, permitindo a integração entre as tecnologias.

Camada de funcionamento do EF:

		UI 			- Interação do Usuário com o aplicativo (Camada mais alta)
Camada de Negócios 	- Classes, Abstração, processamento e tratamento de dados (Camada Intermediária)
  Camada de Dados	- Comunicação com o Banco de Dados (Camada mais baixa)
		|			  	+ O Entity Framework irá abstrair as consultas SQL e facilatará
		V				  a integração com o banco de dados relacional.	
  Banco de Dados	- Conjunto de dados e tabelas agrupados

O entity framework permitirá utilizar todos os comandos SQL e SQLServer, inclusive,
criar tabelas caso elas não existam.
